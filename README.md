# se_project

Creative Work Timestamp & Ownership Registry
A full-stack system for registering, timestamping, and verifying ownership of creative content.
Built using Flask, SQLite, Streamlit, SimHash, and RSA cryptography, the platform enables creators to securely prove originality, detect plagiarism, and generate tamper-proof ownership certificates.
